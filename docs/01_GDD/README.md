# Dg225-Team10_Project_Lobotomy

สรุปเกม (High-level)

- แนวเกม: การ์ดแบตเทิลเชิงกลยุทธ์ มีองค์ประกอบ roguelite — เดินผ่านด่าน 5 ชั้นแล้วเจอบอส
- จุดเด่น: จัดการเด็คขนาดเล็ก (12 ใบ) และทรัพยากรจิตใจ (Sanity %) เพื่อใช้การ์ดต่าง ๆ

Core Loop (ลูปการเล่นหลัก)

- เดินสำรวจแต่ละเทิร์น -> พบมอนสเตอร์/เหตุการณ์ -> เข้าสู่การต่อสู้แบบเทิร์นเบส
- แต่ละเทิร์นดึง/สุ่มการ์ดจากเด็ค และเลือกเล่นการ์ดโดยแลกกับค่า Sanity หรือทรัพยากรอื่นๆ
- สะสม/ฟื้นฟู Sanity จากการชนะมอนสเตอร์หรือไอเท็มระหว่างทาง
- เป้าหมาย: เคลียร์ 5 ชั้น แล้วชนะบอสสุดท้าย

ทรัพยากรหลัก

- Sanity (0–100%): เมตริกสำคัญ แทนพลังจิต/สภาพจิตใจของผู้เล่น
  - ใช้เป็นค่าใช้จ่ายของการ์ดบางประเภท (เช่น 10%/15%)
  - ลดตามเวลา/เทิร์น (Decay) และเพิ่มเมื่อชนะมอนสเตอร์หรือใช้ไอเท็ม
- Cain Energy (ตัวเลือก): เป็นทรัพยากรรองสำหรับสกิลพิเศษหรือเปิดการ์ดกำจัด (ถ้าต้องการ)

เด็คและการ์ด (ตัวอย่าง)

- ขนาดเด็คเริ่มต้น: 12 ใบ (ปรับได้ตามบาลานซ์)
- ประเภทการ์ดหลัก:
  - Attack (โจมตี): ตัวอย่าง "Strike": สร้าง_DAMAGE_ หน่วย ใช้ Sanity 10% (สำเนา 3 ใบ)
  - Defense (กัน): ตัวอย่าง "Defend": ให้เกราะ _DEF_ หน่วย ใช้ Sanity 10% (สำเนา 3 ใบ)
  - Support (ซัพพอร์ต): ตัวอย่าง "Heal" / "Power Up" / "Protection Boost" / "Force Field": ฟื้น HP, เพิ่มพลังโจมตี ช่วยป้องกัน ฯลฯ (1 ใบแต่ละชนิดเป็นค่าเริ่มต้น)
  - Mix / Hybrid: ตัวอย่าง "Counter Attack": โจมตีเป็นจำนวน _X/2_ และรับเกราะ _X/2_ ใช้ Sanity 15% (1–2 ใบ)

ตัวอย่างการ์ด (ค่าเริ่มต้นแนะนำ)

- Strike — Damage: 8, Cost: 10% Sanity, Copies: 3
- Defend — Block: 6, Cost: 10% Sanity, Copies: 3
- Heal — Heal: 10 HP, Cost: 12% Sanity, Copies: 1
- Power Up — +2 Damage next turn, Cost: 8% Sanity, Copies: 1
- Force Field — Reduce incoming damage 50% for 1 turn, Cost: 15% Sanity, Copies: 1
- Counter Attack — Damage: 6 and Gain Block: 6, Cost: 15% Sanity, Copies: 1

ระบบ Sanity (รายละเอียด)

- เมทริกเป็นเปอร์เซ็นต์ (0–100)
- ทุกเทิร์นผู้เล่นจะสูญเสีย Sanity จำนวนหนึ่ง (ตัวอย่าง: 1–5% ต่อเทิร์น ขึ้นกับสถานะหรือสภาพแวดล้อม)
- ชนะมอนสเตอร์จะได้คืน Sanity (ตัวอย่าง: +3–10% ขึ้นกับระดับมอนสเตอร์)
- ถ้า Sanity <= 0%: เกิดผลลบขั้นรุนแรง (เช่น สถานะ Panic — จำกัดการ์ดที่เล่นได้ เพิ่มความเสียหายที่รับ)

Combat Flow (ลำดับการต่อสู้)

1. เริ่มเทิร์น: ลด Sanity decay
2. ผู้เล่นได้การ์ดตามกฎการ์ด (ดึงการ์ด/เติมมือ)
3. ผู้เล่นเล่นการ์ดโดยจ่าย Sanity/ค่าใช้จ่ายอื่นๆ
4. คำนวนผลการ์ด (โจมตี/ป้องกัน/บัฟ/ฮีล)
5. ฝ่ายศัตรูทำการโจมตี/สกิล
6. จบเทิร์น, ตรวจสถานะชนะ/แพ้, มอนสเตอร์ตายให้รางวัล Sanity/ไอเท็ม

ศัตรูและบอส (แนวคิด)

- มอนสเตอร์ปกติ: มีระดับง่าย->ยาก ให้รางวัล Sanity แตกต่างกัน
- มินิบอส: ท้าทายกว่าปกติ มีทริค/สเตตัสพิเศษ
- บอสระดับชั้นสุดท้าย: มีเฟสหลายเฟส ใช้กลไกพิเศษเพื่อทดสอบเด็คผู้เล่น

ความก้าวหน้าและบาลานซ์

- ปรับค่า Sanity decay, รางวัล Sanity, และค่าสถานะการ์ดเป็นหลักสำหรับบาลานซ์
- ติดตามค่าพื้นฐาน (เช่น Damage per Sanity, Block per Sanity) เพื่อให้การตัดสินใจมีความคุ้มค่า

UI / UX (ข้อเสนอ)

- แสดงแถบ Sanity เป็น % ชัดเจน พร้อมเอฟเฟ็กต์เมื่อใกล้หมด
- แสดงจำนวนการ์ดในเด็ค/กองทิ้ง/มือ
- Tooltip บนการ์ดแสดงค่า Cost (Sanity), ผลลัพธ์ และตัวอย่างตัวเลข

งานศิลป์ & เสียง (แนะนำ)

- โทนเกม: ดาร์ก-ไซไฟ ผสมสยองเล็กน้อย (ขึ้นกับธีม Lobotomy)
- เอฟเฟ็กต์เสียงสำหรับการ์ดชนิดต่างๆ (โจมตี/ป้องกัน/ฮีล/เสีย Sanity)

หมายเหตุด้านการออกแบบและการทดลอง

- เลือกค่าเริ่มต้นที่อนุญาตให้ผู้เล่นใช้การ์ด 2–3 ใบต่อเทิร์น ก่อนที่ Sanity จะเริ่มเป็นปัญหา
- ทำระบบอัปเกรดเด็ค/การ์ดในแต่ละรันเพื่อเพิ่มความหลากหลาย
- บันทึกสถิติการเล่น (เช่น ค่าสูญเสีย Sanity เฉลี่ย ต่อการชนะแต่ละชั้น) เพื่อช่วยปรับบาลานซ์
